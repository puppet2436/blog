### 数据类型

1. 标量类型

	> 标量：一次只能存储一个值的变量

	表 2-1 常用标量类型

| 类型         | 描述                                                         |
|--------------|--------------------------------------------------------------|
| int          | +/- 2^31 -1 之间的整数                                       |
| unsigned int | 0 ~ 2^32 之间的整数值                                        |
| float        | +/- 2^24 之间的浮点值                                        |
| double       | +/- 2^31 之间的浮点值                                        |
| long         | 根据芯片架构的不同，大小从32位到64位的指数值                 |
| long long    | 64位整数值                                                   |
| char         | 单个字符，通过整形表示                                       |
| BOOL         | 布尔值，YES or NO                                            |
| NSInteger    | 在 32 位机器上，和 int 相同，在 64 位机器上，范围是 +/- 2^32 |
| NSUInteger   | 在32位机器上和 unsigned int 相同                             |
